Introducao

Uma estrutura condicional define um bloco de operações de acordo com uma escolha. Existem quatro estruturas condicionais:

  • if;
  • else;
  • else if;
  • switch.

Estruturas Condicionais

O if

A estrutura segue a seguinte sintaxe: 

                        if(bloco de condição){ bloco de operações; }

A condição pode assumir um valor verdadeiro ou falso. O bloco de operações só é executado se a condição for verdadeira. Um bloco de operações pode conter outros blocos de operações, ou seja, pode haver estruturas de seleção aninhadas.

O else

A estrutura segue a seguinte sintaxe: 

                        if(bloco de condição){ bloco de operações; }
                        else{ bloco de operações; }

Uma versão mais completa da cláusula if, contém blocos de operações tanto na parte verdadeira quanto na falsa. O bloco de operações associado a condição falsa começa com a cláusula else.

O else if

A estrutura segue a seguinte sintaxe: 

                        if(bloco de condição){ bloco de operações; }
                        else if(bloco de condição){ bloco de operações; }

Verifica uma nova condição caso a anterior tenha sido falsa. Caso a anterior tenha sido verdadeira ele não é verificado e executado.

O switch case

A estrutura segue a seguinte sintaxe: 

                        switch(expressão integral){
                            case valor 1:
                            //bloco de operações
                            break;
                            case valor n:
                            //bloco de operações
                            break;
                            default:
                            //bloco de operações
                            break;
                        }

Uma alternativa multi-escolha para o if. Dispensa o uso de vários if's seguidos e consequentemente abaixa a complexidade do código.

Exemplos

if, else if e else 

                        if(X > Y){
                            printf("O maior eh X");
                        } else if(Y > X){
                            printf("O maior eh Y");
                        } else {
                            printf("Os numeros sao iguais");
                        }

switch case 

                        switch(opcao_menu){
                            
                            case 0:
                            printf("voce saiu do menu")
                            return 0;
                            break;
                            
                            case 1:
                            funcao_1();
                            break;
                            
                            case 2:
                            funcao_2();
                            break;

                            case 3:
                            funcao_3();
                            break;

                            default:
                            funcao_default();
                            break;
                        }
